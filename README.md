# Nathan GRACIA — Portfolio

Portfolio personnel one-page. Dev web full stack & Data Scientist.

**Live :** https://nathangracia.github.io/portfolio

## Stack

HTML/CSS/JS vanilla — pas de framework, pas de bundler. Un seul fichier `index.html` + une photo.

## Structure

```
portfolio/
├── index.html        # tout le site (HTML + CSS + JS)
└── assets/
    └── photo.jpg     # photo de profil
```

## Sections

Hero · À propos · Skills · Projets · Loisirs · CV · Contact

## Features

- Dark mode (bouton bas droite)
- Scroll animations (IntersectionObserver)
- Nav avec backdrop-blur au scroll
- Responsive mobile (< 820px)
- Bilingue FR/EN (partiel, extensible)

## Déploiement

GitHub Pages : Settings > Pages > Source `main` / `/ (root)`
